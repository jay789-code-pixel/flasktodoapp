# Project Statement: Flask Todo App

## Objective

To develop a lightweight, easy-to-use todo management web application using Flask. The aim is to showcase practical web development skills, CRUD operations, and UI design in Python, enabling users to efficiently organize daily tasks.

## Scope

- User can add, modify, complete, and delete todo items.
- Todos are stored in a local database (SQLite).
- Application has a clear, responsive frontend using Bootstrap.
- The app provides basic session handling for state persistence.

## Motivation

Managing to-do lists is a common necessity. This app is built to meet that need while serving as a demonstration of Flask fundamentals and full-stack application development in Python. The project is designed as a learning tool, code example, and a basis for future extensions.
